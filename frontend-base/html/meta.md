# meta总结

```html
 <!-- 声明文档使用的字符编码 -->
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<!-- 优先使用 IE 最新版本和 Chrome内核渲染 -->
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1" />
<!-- 移动设备添加 viewport -->
<meta name="viewport" id="viewport" content="user-scalable=no, width=device-width, initial-scale=1, maximum-scale=1, viewport-fit=cover, target-densitydpi=device-dpi" />
<!--telephone=no就禁止了把数字转化为拨号链接-->
<meta name="format-detection" content="telephone=no" />
<!--安卓 UC 浏览器下，会偶尔出现字体突然变大的情况-->
<meta name="wap-font-scale" content="no">
<!--删除默认的苹果工具栏和菜单栏-->
<meta name="apple-mobile-web-app-capable" content="yes" />
<!--控制状态栏显示样式-->
<meta name="apple-mobile-web-status-bar-style" content="black" />
<!-- QQ浏览器强制全屏 -->
<meta name="x5-fullscreen" content="true" />
<!-- UC浏览器强制全屏 -->
<meta name="full-screen" content="yes" />
```
